# vrf_rng

### Commands

- Install dependencies: `pnpm install`
- Compile contracts: `pnpm build`
- Typegen: `pnpm typegen`
- Deploy smart contract: `pnpm deploy_contract`
- Try out the contract: `pnpm test_contract`
